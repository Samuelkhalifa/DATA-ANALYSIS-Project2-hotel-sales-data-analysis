## [Problématique : Analyser les ventes d'un hôtel sur 3 ans]


### &#127919; Objectifs
* Récupérer, préparer et analyser les données.
* Importer le dataset nettoyé dans (`MySQL`) installé sur une machine virtuelle (`parallels`)
* Ecrire les requêtes nécessaires et connecter le résultat à l'outil de dashboarding (`Power Bi`)
* Construire un dashboard permettant de répondre à la demande d'analyse.
